Este projeto consiste em um sistema bancário desenvolvido em Python, com foco em simular operações reais de um banco. A aplicação permite gerenciar contas, realizar transações financeiras e acompanhar o histórico de operações.

O sistema foi desenvolvido utilizando conceitos de programação orientada a objetos (POO), visando organização, escalabilidade e clareza no código.

Funcionalidades
✔ Criação de contas bancárias
✔ Depósito de valores
✔ Saque com validação de saldo
✔ Transferência entre contas
✔ Histórico de transações
✔ Listagem de contas cadastradas
✔ Identificação única para cada conta
